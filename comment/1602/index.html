Gimp 2.8 konečně vydán! (aktualizováno)

3. May 2012 - 18:35 — Lukáš

Vývojáři na Google+ oznámili, že po 3 a půl letech práce vydali finální verzi - Gimp 2.8. Změn je hodně viz. Release notes. Nicméně na port pro Windows, nebo Mac OS X si musíme ještě počkat

Verzi pro Windows stahujte zde: http://sourceforge.net/projects/gimp-win/files/GIMP%20%2B%20GTK%2B%20%28stable%20release%29/GIMP%202.8.0/gimp-2.8.0-setup.exe/download
